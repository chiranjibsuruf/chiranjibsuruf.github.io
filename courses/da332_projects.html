Project

A data visualization course project proves you can turn messy data into clear, decision-ready stories—not just run analyses. It gives you a portfolio piece that shows real-world impact: insight discovery, communication, and dashboard-quality execution.

Getting Started

List of Probable Project Topics

  • Interactive Sales Dashboard (region/product/time filters; KPIs + trends)
  • COVID/Health Trends Story (cases, vaccination, mortality; annotations + narrative)
  • Climate Change Visual Report (temperature anomaly, CO₂, extreme events; small multiples)
  • Stock/Portfolio Performance Analyzer (returns, risk, drawdowns; interactive charts)
  • Customer Churn Exploration (cohort charts, feature comparisons, churn drivers)
  • Marketing Funnel Visualization (conversion drop-offs, channel performance, attribution view)
  • Supply Chain / Inventory Monitor (stockouts, lead times, service levels, heatmaps)
  • HR Analytics Dashboard (hiring, diversity, attrition, compensation bands—ethics-aware)
  • Public Transport / Mobility Map (routes, delays, demand hotspots; geo-visualizations)
  • Food Delivery / Restaurant Insights (ratings, prep time, cuisine clusters, map-based demand)
  • Education Analytics (student performance, engagement, topic mastery; drill-down views)
  • Financial Statement Visualizer (income statement trends, margins, waterfall charts)
  • E-commerce Product Analytics (basket analysis visuals, price vs rating, category treemaps)
  • Sports Performance Dashboard (team/player stats, shot charts, win probability trends)
  • Network Visualization Project (social graph, citation network, or collaboration map)
  • Text Visualization (topic modeling + word shifts + sentiment timelines)
  • Data Quality & Missingness Report (completeness heatmaps, anomaly flags, audit summary)
  • “One Dataset, Many Charts” Gallery (best chart choice for each question; design critique)
